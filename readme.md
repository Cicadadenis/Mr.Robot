# Mr.Robot
Mr.Robot Служба коротких сообщений.


### Requirements
• Python 3.x

#### Установка и использование Mr.Robot
```bash
git clone https://github.com/bednakovdenis/Mr.Robot
cd Mr.Robot
python -m pip install -r requirements.txt
python3 Mr.Robot
```
